# ITSD85: GitHub Activity


### Creating a classic personal access token
1. Once logged-in to GitHub, visit [github.com/settings/tokens](github.com/settings/tokens).
2. Click **Generate new token › Generate new token (classic)**
3. Write any **Note** about the new token 
    Ex: “itsd85”
4. Click **Generate token** button
5. Note of the personal access token because you will be using this
as a **password** later
